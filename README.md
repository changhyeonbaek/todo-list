# Todo App

React와 Redux Toolkit을 활용하여 제작된 간단한 Todo 리스트 애플리케이션입니다. 사용자는 할 일을 추가하고, 완료 상태를 토글하며, 불필요한 항목을 삭제할 수 있습니다. 깔끔한 UI와 반응형 스타일링으로 사용자 경험을 개선했습니다.

## 주요 기능
- 새로운 할 일 추가
- 할 일 완료/미완료 상태 토글
- 할 일 삭제
- 반응형 UI와 애니메이션 효과
